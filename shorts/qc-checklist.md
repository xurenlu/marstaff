# 动漫短剧 — 单集质检清单（QC）

> 每集成片或每轮 `video_story_workflow_create` 完成后，在聊天中逐项过一遍；不通过则 **先改圣经或分镜表**，再重建工作流，避免仅微调 prompt 碰运气。

## 1. 区分度（脸盲）

- [ ] 遮住对白与字幕，观众能否仅凭 **外形** 区分核心角色 A/B/C？
- [ ] 每个含角色的 scene 是否都嵌入了 **完整角色标签句**（或等价复述）？
- [ ] 同框镜头是否滥用「双人大正面特写」？（若有多，标签句是否足够长、站位是否固定？）

## 2. 连贯性（穿帮）

- [ ] 相邻镜之间 **服装** 是否一致（除非剧情换装）？
- [ ] **光线方向** 与 **时间感**（日/夜）是否连续？
- [ ] **站位**（左/右）与 continuity 描述是否一致？
- [ ] **道具** 在谁手中、是否在上一镜出现过，无凭空消失？

## 3. 节奏与可生成性

- [ ] 是否有某一镜 **信息过载**（一镜内塞太多事件）？应拆成多 scene。
- [ ] 每 scene **时长** 是否落在单次模型上限内？
- [ ] 全局 **style** 与各 scene 是否无冲突（禁止一镜水墨一镜写实）？

## 返工顺序（建议）

1. 修改 `01_characters.md` 标签句或锚点（若区分度失败）。
2. 修改 `ep##_storyboard.md` 的 continuity / 拆分过长的镜。
3. 必要时回调 `00_series_bible.md` 的风格句或视觉母题。
4. 重新调用 `video_story_workflow_create`（勿用 `pipeline_create` 手搓拼接）。
